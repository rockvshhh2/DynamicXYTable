DynamicXYTable : 
David.Huang
DynamicXYTable is a multi-functional 'TreeView Table plug-ins.' based on jQuery. 

How do use :
$("Div Selector").BBTree({Options});


API Documents : 
Option : {
X: [
    {
    Text: string, // Node's  Text
    Key: string, // Node's Td Tag Id (ext : td_Key )
    Children: Array // Array of child nodes.
    }
],
Y: [
    {
    Text: string, // Node's  Text
    Key: string, // Node's Td Tag Id (ext : td_Key )
    Children: Array // Array of child nodes.
    }
]
}
